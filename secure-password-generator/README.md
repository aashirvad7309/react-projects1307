# 🔐 Secure Password Generator
A clean & customizable password generator built with React and Tailwind CSS.

## 🚀 Features
- Set password length
- Copy to clipboard
- Auto strength indicator
- Fully responsive UI

## 📦 Tech Stack
React • TailwindCSS • Vite
